# MISCLIPS

Plataforma open source para análisis deportivo que combina video, datos tipo Excel y compartición inteligente de jugadas.

---

## 🚀 Descripción

MISCLIPS es una aplicación web progresiva (PWA) que permite trabajar con videos deportivos de forma eficiente, enfocándose en **momentos específicos (clips)** en lugar de archivos completos.

Los usuarios pueden cargar partidos completos, definir jugadas clave y compartirlas mediante listas de reproducción accesibles incluso para usuarios sin cuenta.

Además, MISCLIPS integra datos provenientes de herramientas externas como Excel, permitiendo combinar **video + datos + análisis** en una sola plataforma.

---

## 🎯 Objetivo

Facilitar el análisis y la distribución de contenido deportivo mediante:

* Acceso rápido a jugadas específicas
* Edición flexible tipo Excel
* Compartición sencilla con otros usuarios
* Reducción de uso de almacenamiento y ancho de banda

## 🌐 Funcionamiento

### 📹 Video

* Subida de partidos completos
* Definición de clips sin recortar archivos físicamente
* Creación de listas de reproducción compartibles

### 🏷️ Etiquetas

* Asignación de etiquetas simples a cada momento (ej: saque, recepción, error, punto)
* Sistema de etiquetado rápido, pensado para no interrumpir el flujo de análisis

---

## 📊 Integración con Excel (QuickMark Volleyball)

MISCLIPS está diseñado para integrarse con flujos de trabajo existentes:

* Exportación de datos desde Excel (QuickMark Volleyball)
* Importación en la PWA
* Asociación de datos con momentos de video

### ✏️ Edición tipo Excel

Dentro de MISCLIPS se podrá:

* Editar datos fácilmente
* Corregir errores rápidamente
* Crear nuevas columnas personalizadas
* Filtrar momentos según distintos criterios

Esto convierte a MISCLIPS en una herramienta híbrida:
**video + base de datos editable**
